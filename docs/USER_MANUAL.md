## 1. 開始使用

開啟瀏覽器，前往應用程式網址。若尚未登入，系統會自動導向登入頁面。

**建議首次使用流程：**

```
註冊帳號 → 建立帳戶 → 建立基金 → 開始記帳
```

---

## 2. 登入與註冊

### 註冊新帳號

1. 進入登入頁面，點擊上方 **「註冊」** 分頁。
2. 輸入你的 **電子郵件** 與 **密碼**（至少 6 個字元）。
3. 點擊 **「建立帳號」**。
4. 成功後自動登入並跳轉至儀表板。

### 登入現有帳號

1. 確認分頁切換至 **「登入」**。
2. 輸入電子郵件與密碼。
3. 點擊 **「登入」**。
4. 成功後跳轉至儀表板。

> **提示：** 若輸入有誤，頁面下方會以紅色顯示錯誤訊息。

---

## 3. 儀表板總覽

登入後進入主頁面（`/dashboard`），畫面分為四個功能卡片區塊：

| 卡片區塊 | 功能說明 |
|---------|---------|
| 建立帳戶 | 新增用來存放資金的帳戶（例如：薪轉戶、儲蓄戶） |
| 建立基金 | 新增用來分配用途的基金（例如：旅遊基金、緊急備用金） |
| 新增交易 | 記錄儲值或支出 |
| 交易查詢 | 查詢與篩選交易紀錄 |

右上角按鈕：
- **「查看基金圖表」** — 跳轉至圖表頁
- **「登出」** — 結束工作階段

---

## 4. 建立帳戶

「帳戶」代表資金實際存放的地方（例如銀行帳戶）。

**步驟：**

1. 在「建立帳戶」卡片中，輸入 **帳戶名稱**（例如：玉山薪轉）。
2. 從下拉選單選擇 **帳戶類型**：
   - **薪轉戶** — 日常收入來源帳戶
   - **儲蓄戶** — 長期存放帳戶
3. 點擊 **「新增帳戶」**。
4. 成功後，帳戶名稱出現在下方列表中。

---

## 5. 建立基金

「基金」代表資金的使用用途（例如：旅遊、日常開銷）。

**步驟：**

1. 在「建立基金」卡片中，輸入 **基金名稱**（例如：旅遊基金）。
2. 輸入 **循環日**（數字 1–31），代表每月補充資金的起始日期。
3. 點擊 **「新增基金」**。
4. 成功後，基金名稱與循環日出現在下方列表中。

> **範例：** 循環日設為 `5`，表示每月 5 日為該基金的月結算起點。

---

## 6. 新增交易

記錄資金的儲值或支出。

**步驟：**

1. 在「新增交易」卡片的下拉選單中，選擇 **基金** 與 **帳戶**。
2. 選擇 **操作類型**：
   - **儲值 (TOP_UP)** — 將資金加入基金
   - **支出 (EXPENSE)** — 從基金扣除資金
3. 輸入 **金額**（支援小數，最小 0.01）。
4. 輸入 **備註**（選填，例如：早餐、7-11）。
5. 點擊 **「送出交易」**。
6. 成功後顯示綠色提示，表單自動清空。

> **注意：** 基金餘額 = 所有 TOP_UP 總和 - 所有 EXPENSE 總和，系統即時計算，不另行儲存。

---

## 7. 查詢交易紀錄

篩選並瀏覽指定基金的歷史交易。

**步驟：**

1. 在「交易查詢」卡片中，從下拉選單選擇 **基金**。
2. 選擇 **交易類型篩選**（選填）：
   - 全部類型
   - TOP_UP（僅儲值）
   - EXPENSE（僅支出）
   - ADJUST（僅調整）
3. 設定 **每頁筆數**（預設 10，最多 100）。
4. 點擊 **「查詢」**。
5. 下方表格顯示結果：

| 欄位 | 說明 |
|------|------|
| 類型 | TOP_UP / EXPENSE / ADJUST |
| 金額 | 交易金額 |
| 備註 | 交易說明（若有填寫） |
| 時間 | 交易發生時間 |

**翻頁：**
- 點擊 **「下一頁」** 或 **「上一頁」** 瀏覽更多紀錄。
- 頁碼顯示格式：`第 X 頁 / 共 Y 筆`。

## 8. 基金圖表

以月份為單位，視覺化呈現基金的收支走勢。

**前往方式：** 點擊儀表板右上角 **「查看基金圖表」**。

**操作步驟：**

1. 從下拉選單選擇要查看的 **基金**。
2. 點擊 **「更新圖表」**。
3. 圖表呈現：
   - **長條圖（藍色）** — 每月儲值（TOP_UP）金額
   - **折線圖（紅色）** — 每月支出（EXPENSE）金額
   - 滑鼠移至圖表上可顯示精確數值 tooltip。

**返回：** 點擊左上角 **「返回儀表板」**。

---

## 9. 主題切換

應用程式支援 **淺色** 與 **深色** 兩種主題。

- 點擊右上角圓形按鈕切換主題：
  - ☀️ — 淺色模式
  - 🌙 — 深色模式
- 主題偏好會儲存於瀏覽器，下次開啟時自動套用。

---

## 10. 登出

- 點擊任意頁面右上角 **「登出」** 按鈕。
- 系統清除登入狀態並跳轉至登入頁面。

## 11. 常見問答

**Q: 基金餘額在哪裡查看？**
A: 目前餘額由交易紀錄加總計算。你可以在交易查詢中查看所有 TOP_UP 與 EXPENSE，手動加總或參考圖表頁面的累積趨勢。

**Q: 帳戶和基金有什麼差別？**
A: **帳戶** 是資金存放的地方（實體銀行帳戶），**基金** 是資金的用途分類（例如：旅遊、日常）。每筆交易都需同時指定帳戶與基金。

**Q: 記錯金額怎麼辦？**
A: 系統採用「只增不刪」的帳本設計。若記錯，請新增一筆 **ADJUST** 類型的調整交易來修正（目前 ADJUST 需透過後端 API 操作，前端 UI 尚未提供此選項）。

**Q: 忘記密碼怎麼辦？**
A: 目前版本尚未提供密碼重設功能，請聯繫系統管理員。

**Q: 支援哪些瀏覽器？**
A: 建議使用最新版 Chrome、Firefox、Edge 或 Safari。
